git基本使用

发表于 2018-03-26

初始化Git文件夹(新建空白文件夹，进入文件夹)

git init

添加到缓存区,使用如下命令:

git add -A

实际提交改动commit文件:

git commit -m "msg"

添加远程仓库(如果没有):

git remote add origin <server>
git remote add origin git@github.com:zhengzhiyuu/MD_backups.git

向远程仓库推送改动:

git push -u origin master
推荐阅读此教程，赞!
